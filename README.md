# Random-forest-classifier-with-quantum-model

Basandoci sul lavoro del “Canadian Institute for Cybersecurity (CIC)” vogliamo verificare se il modello quantistico è migliore del modello classico.
L’idea è quella di testare dei modelli quantistici e verificare efficienza e complessità computazionale.
